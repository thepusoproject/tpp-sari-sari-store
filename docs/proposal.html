Education 5.0 Roadmap

This Education 5.0 roadmap (for DepEd DO 001, s. 2026) grounds the Programming para sa Lahat CLI pilot inside the Convergence Theory of Learning (CTL) and its IPOO spine (Input → Process → Output → Outcome). Every milestone scales across ALS divisions via DIWA (the purpose-built learning portal) while keeping human mentors in the loop.

DepEd DO 001 anchors

  • Personalized pacing: Initial diagnostics (ABL + FLT) and Individual Learning Agreements (ILA) set the sequence, pace, and depth per learner.
  • Differentiated delivery: CTL mandates multi-modal mentoring (modular, online/mobile, face-to-face, radio/TV, blended) so Region VIII, CAR, and partner hubs can mix connectivity options.
  • Recognition + evidence: Every learner keeps a Record of Progress with Recognition of Prior Learning (RPL) artefacts plus a portfolio of outputs (21st-century skills, values).
  • Teacher role: Mentors shift from content delivery to learning designers who use DIWA analytics + oral checks to guide BYOB/WBL outcomes.

IPOO learning flow

  1. Input: DIWA portal captures context (ABL, FLT, socio-economic data, device readiness) and seeds each learner’s ILA.
  2. Process: Vibe Coding UX + AI scaffolds adapt module order, trigger check-ins, and feed mentors with “next best” nudges.
  3. Output: GitHub-based portfolios, wellbeing reflections, and Studio artefacts become the competency evidence required by DO 001.
  4. Outcome: Solidarity, Integrity, and Creativity indicators connect to BYOB/WBL placements, employment data, and community projects.

Recent pilot improvements

  • Concept checks + bot triage: Written reflections, placeholder validation, and random question prompts per PR.
  • Roster automation: Cloudflare Worker handles enrollment issues without exposing secrets.
  • Gradeboard + Docs hub: Live data on Grades + architecture/roadmap pages for stakeholders.
  • Integrity guardrails: Oral-check label, structural concept checks, and future LLM triage hooks.

Milestones

Now shipping

Milestone 0 — CLI + Instructor Automations

  • Programming para sa Lahat curriculum + sari-sari-store project.
  • Automated grading, Gradeboard, concept prompts, oral-check workflow.
  • Cloudflare roster worker + GitHub-only submission pipeline.

Purpose: prove the coding-first stack, collect mastery data, and surface insights for Education 5.0 layers.

Milestone 1 — Adaptive & Wellbeing Layer

  • Rules-based “next best activity” feed with teacher overrides.
  • Wellbeing check-ins, mentoring notes, “needs attention” flags.
  • Studio authoring UX + ALS-aligned portfolio exports.
  • Optional LLM concept triage (auto-score reflections, flag anomalies for mentors).

Matches proposal pillars 1 & 2: AI-augmented learning paths + holistic learner dashboard with privacy guardrails.

Milestone 2 — Immersive Projects & Offline PWA

  • VR/AR mini-simulations (2–5 modules) tied to Studios.
  • Offline-first PWA (draft + sync) for low-connectivity learners.
  • Community/values evidence capture (solidarity, creativity, integrity).

Matches pillar 3 + CTL IPOO: experiential learning with reflection + values baked in.

Milestone 3 — Smart Spaces Pilot (Optional)

  • Small IoT deployment (1–2 learning hubs) for environment + engagement data.
  • Strict consent + governance, only after LMS impact is proven.
  • Feedback loop into wellbeing dashboard + program decisions.

Matches pillar 4: connect physical spaces without compromising privacy.

Key acronyms (spelled out once)

  • CTL: Convergence Theory of Learning (Fr. Benigno P. Beltran’s Education 5.0 pedagogy).
  • IPOO: Input → Process → Output → Outcome framework used for planning + reporting.
  • ALS: Alternative Learning System (DepEd program serving out-of-school youth/adult learners).
  • DIWA portal: “Design Intelligence for Wisdom and Agency” learning mesh (ABL, FLT, ILA, analytics).
  • ABL / FLT: Assessment of Basic Literacy / Functional Literacy Test.
  • ILA: Individual Learning Agreement signed by learner + implementer.
  • RPL: Recognition of Prior Learning (formal/non-formal/informal).
  • BYOB / WBL: Be Your Own Boss micro-enterprise track / Work-Based Learning placements.
