Para sa Pilipino

Programming para sa lahat.

Ang course ay base sa Python for Everybody ni Prof. Charles Severance. Filipino-English translation ng Data Engineering Pilipinas at Code.Sydney.

Ang Taglish Py4E ay micro-learning na naka-angkla sa tunay na developer workflow: GitHub branches, pull requests, automated grading, at mentoryo mula sa komunidad. Perfect para sa mga gustong mag-level up habang nagtatrabaho o nag-aaral.

  • Taglish explainers + Filipino context
  • Real-time GitHub Actions feedback
  • Mastery-based unlocks (≥80%)
  • Community-driven coaching

Ano’ng meron sa track?

Isang evolving case study: ang TPP Sari-Sari Store CLI. Bawat module ay micro-upgrade sa tunay na store ops.

M01 · I/O at Variables

Unang buhos: welcome screen, basic inventory input, at pag-print ng resibo sa Taglish prompts.

🕑 2–3 hrs ✨ Mastery gate: 80%

M02 · Conditionals

Promo logic, senior citizen discount, at anti-palpak validation gamit ang if/elif/else.

🕑 3 hrs 📦 Inventory rules

M03 · Functions

Refactor para hindi spaghetti: helper functions, docstrings, at mas malinaw na flow.

🕑 3–4 hrs 🧠 Code health

M04 · Loops + Files

Hands-on sa stock counts, CSV logs, at repeatable tasks—parang tunay na tindahan.

🕑 4 hrs 🗂 Data hygiene
